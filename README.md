# awesome-k8s-installer
A curated list of awesome Kubernetes installers

* [kubekey](https://github.com/kubesphere/kubekey/) Provides a flexible, rapid and convenient way to install Kubernetes only, both Kubernetes and KubeSphere, and related cloud-native add-ons. It is also an efficient tool to scale and upgrade your cluster. ![GitHub Repo stars](https://img.shields.io/github/stars/kubesphere/kubekey?style=social)
